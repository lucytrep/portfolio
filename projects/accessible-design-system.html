Overview

Accessible defaults that scale across teams

This project focused on building an accessible design system that teams could adopt quickly—reducing inconsistency while raising the baseline for inclusive UI. I balanced component coverage with documentation quality so the system was practical to implement, not just polished in Figma.

The system included tokens, component patterns, and guidance designed to prevent common accessibility issues before they ship.
